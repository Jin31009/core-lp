# RA-SS Slide Archive v1.0

## プロジェクト概要
RA-SSスライド群の最終成果を、再利用可能な形で保存・構造化したアーカイブです。
本アーカイブは「内容を改変せず、保存と構造整理のみ」を方針としています。

## 構成（スライドの流れ）
1. 表紙
2. 観測設計（05〜08）
3. CASE①〜③
4. 可能性への転換（14/17）
5. ACE-X（15/17）
6. RA-SS（17/17）
7. DATA（302件分析結果）
8. 結論

## 各章の役割
- `01_cover`：導入・世界観提示
- `02_observation_design`：観測条件の設定と構造理解の入口
- `03_case_analysis`：ケース分解による適用確認
- `04_transition`：理解から可能性へ転換
- `05_acex`：介入モデル（ACE-X）の提示
- `06_rass`：RA-SS運用フレーム
- `07_data`：302件分析結果
- `08_conclusion`：総括

## 使用モデル（RA-SS）
Relational Analysis & Structure System（RA-SS）
