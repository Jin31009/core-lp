# Production Rules

## 表紙
- 目的：世界観提示
- 情報密度：中
- キャラ露出：4人全員
- トーン：デフォルメ3D（トイモデル風）、温かい
- カメラ：正面基準（必要に応じて微調整可）

## 扉
- 目的：概念提示
- 情報密度：低
- キャラ露出：最小
- トーン：静か、余白

## 本文
- 目的：理解プロセス
- 形式：4〜5コマ
- キャラ露出：役割に応じて
- トーン：対話・流れ（デフォルメ3D固定）
- カメラ運用：
  - キャラ・空間・小物は固定
  - カメラ距離/位置/角度は意図に応じて変更可
  - 同一空間で「見え方だけを変える」ことを優先
  - サムネイル表示・極小配置はNG

## 質感・光（全フォーマット共通）
- 頭身：2〜3頭身固定
- 顔：baseline完全一致（変更禁止）
- シルエット：丸みを維持
- 表面：マット（プラスチック/フィギュア風）
- 光：柔らかい映画光

## 禁止（全フォーマット共通）
- 写実3D（別人化）
- 2Dイラスト化
- 顔の再設計
- 固有スタイル名での指定

## CASE
- 目的：実務証明
- 情報密度：高
- キャラ露出：最小（補助）
- トーン：論理・検証
