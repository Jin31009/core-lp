# Always Baseline (毎回これを使う)

このファイルは、RA-SS制作時に**毎回最初に参照する固定基準**。
画像生成・差分修正・スライド差し替えの前に、必ずこの基準を使う。

## 3) 毎回の開始手順
1. `ALWAYS_BASELINE.md` を読む
2. `BASELINE_REFERENCES.md` を読む
3. 対象スライドの直前/直後スライドを確認（連続性チェック）
4. 生成・修正に着手

## 3.5) スタイル固定（毎回）
- デフォルメ3D（トイモデル風）
- 頭身は2〜3頭身
- 顔はbaseline完全一致（変更禁止）
- 丸みのあるシルエット
- 表面はマット（プラスチック/フィギュア風）
- ライティングは柔らかい映画光

## 4) 禁止
- 上記基準から外れた顔・頭身・配色・空間構造
- 新規スタイルへのジャンプ
- 一時生成物を公式基準へ昇格
- 写実3D化
- 2Dイラスト化
- 顔の再設計
- 固有スタイル名での誘導
